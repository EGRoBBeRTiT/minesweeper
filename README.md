# minesweeper

Тестовое задание

Minesweeper - браузерная игра в сапёра

## Используемые технологии

- Canvas
- TypeScript

## Ключевые особенности

- Поддержка больших полей до **100 000 000** ячеек
- Поддержка светлой и темной тем
- Автоматическое сохранение игры
- Анимации
